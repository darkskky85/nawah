# قسم الأخبار التلقائي — دليل التشغيل

قسم أخبار ذكاء اصطناعي يتحدّث **تلقائياً** كل 6 ساعات عبر GitHub Actions. السكربت يجلب
أحدث الأخبار من مصادر تقنية معروفة، ويصيغ لكل خبر **عنواناً وملخّصاً عربياً أصلياً** عبر
Claude API (لا نسخ — إعادة صياغة)، ثم ينشر التحديث على موقعك.

## الملفات المضافة
- `news.html` — صفحة الأخبار (مربوطة في قائمة كل صفحات الموقع).
- `assets/news-data.js` — بيانات الأخبار (يولّدها السكربت؛ لا تعدّلها يدوياً).
- `scripts/fetch_news.py` — سكربت الجلب والصياغة بالعربية.
- `.github/workflows/news.yml` — جدولة التشغيل التلقائي.

## شرط التشغيل: الموقع على GitHub
الأتمتة تعمل عبر GitHub Actions، فلا بد أن يكون الموقع في مستودع GitHub
(ويُنشر منه إلى Vercel — وهو الإعداد الأكثر شيوعاً). إن لم يكن كذلك:
1. أنشئ مستودعاً جديداً على github.com.
2. ارفع ملفات الموقع إليه.
3. في Vercel: اربط المشروع بالمستودع (Import Git Repository) بدل الرفع اليدوي.

## خطوة الإعداد الوحيدة: مفتاح API (مرة واحدة)
السكربت يحتاج مفتاح Anthropic API ليصيغ الأخبار بالعربية:
1. احصل على مفتاح من `https://console.anthropic.com` (تبويب API Keys).
2. في مستودع GitHub: **Settings → Secrets and variables → Actions → New repository secret**.
3. الاسم: `ANTHROPIC_API_KEY` — القيمة: مفتاحك. احفظ.

> المفتاح يُحفظ مشفّراً في GitHub ولا يظهر في الكود أبداً. لا تضعه في أي ملف.

## التشغيل
- **تلقائياً:** كل 6 ساعات بلا تدخّل.
- **يدوياً للتجربة:** تبويب **Actions** في المستودع → «تحديث أخبار الذكاء الاصطناعي» → **Run workflow**.
- بعد أول تشغيل ناجح، افتح `news.html` وستجد الأخبار ظهرت.

## التكلفة
الصياغة تستخدم نموذجاً سريعاً واقتصادياً (Haiku). كل تشغيل يصيغ حتى 12 خبراً،
والتكلفة التقديرية أقل من سنت واحد لكل تشغيل (بضعة سنتات شهرياً). راجع تسعير
Anthropic الحالي للتأكّد.

## ضبط الإعدادات (اختياري)
في `scripts/fetch_news.py` أعلى الملف يمكنك تعديل:
- `MAX_ITEMS` — عدد الأخبار المعروضة (افتراضي 12).
- `FEEDS` — قائمة المصادر (أضف/احذف خلاصات RSS).
- `MODEL` — نموذج الصياغة.
وفي `.github/workflows/news.yml` يمكنك تغيير `cron` لتكرار مختلف
(مثلاً `0 */3 * * *` كل 3 ساعات).

## ملاحظات
- بعض المصادر قد تحجب الجلب الآلي أحياناً؛ السكربت يتخطّاها ويكمل بالبقية، فلا يتعطّل القسم.
- إذا لم تُجلب أخبار جديدة، يُبقي السكربت آخر نسخة كما هي (لا يفرّغ الصفحة).
- الملخّصات أصلية بصياغة عربية، وكل بطاقة تحيل إلى المصدر — احتراماً لحقوق النشر.
- `scripts/news-seen.json` يُنشأ تلقائياً ليتجنّب تكرار صياغة نفس الخبر.
